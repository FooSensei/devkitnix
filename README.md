# devkitnix

Collection of devkitpro toolchains compatible with Nix.
To install toolchain, use following:

```
git clone https://github.com/knarkzel/devkitnix
cd devkitnix
nix-build <toolchain>.nix
```

## devkitARM

- Game Boy Advance
- Nintendo DS
- Nintendo 3DS
- GP32
- GP2X

## devkitPPC

- GameCube
- Wii
- Wii U
