# devkitnix

Collection of devkitPro toolchains using Nix.
To install a toolchain, use following:

```
git clone https://github.com/zig-homebrew/devkitnix
cd devkitnix
nix-build <toolchain>.nix
```

## devkitARM

- Game Boy Advance
- Nintendo DS
- Nintendo 3DS
- GP32
- GP2X

## devkitPPC

- GameCube
- Wii
- Wii U

## devkitA64

- Nintendo Switch
